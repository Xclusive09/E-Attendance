# E-Attendance System

## Project Overview
The E-Attendance System is a web-based attendance management system that allows users to check in and view their attendance records. Admins can manage user records and view attendance data.

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap
- SweetAlert2
- Notiflix
- Fetch API
